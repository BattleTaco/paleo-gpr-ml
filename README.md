# paleo-gpr-ml

A research-oriented machine learning project exploring non-invasive fossil prospecting using ground-penetrating radar (GPR) and downstream fossil analysis through computer vision.

## Project Goal
This project investigates whether machine learning can help detect fossil-like subsurface anomalies from GPR data and support fossil identification or segmentation from recovered images or CT scans.

## Research Directions
- GPR anomaly detection for fossil prospecting
- Synthetic subsurface data generation
- Fossil image classification
- Fossil CT segmentation

## Status
Early-stage research setup, literature review, and repository initialization.
